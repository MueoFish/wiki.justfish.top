# 获取API

## 无绑定手机抓取


访问OpenAI官网: https://openai.com/ 并登录, 选择 `API选项`


打开 `控制台` 并选择 `网络` 栏,  刷新页面，找到 `login` 请求 ，查看详情

图中红框位置即为API ，完整格式如下
```
sess-Xxxxxxxxxxxxxxxxxxxxxxxxxxxx
```
